# Mullus-Surmuletus

In this repository you can find the R scripts used in the published article "Quantitative histology for maturity staging : Timing is everything" (https://www.sciencedirect.com/science/article/pii/S0165783626001281)

## Scripts:
- Script_dif_3_agents.Rmd : R script in Markdown, uses data from the Calibration dataset folder found in either Zenodo repositories (https://zenodo.org/records/8060356 or https://zenodo.org/records/8060309). This script includes all calibration readings results used for the Inter-agent calibration in stereology.
- Script Article.Rmd : R script in Markdown that uncludes analyses and results for the Cellular homogeneity check, Stereology results and Hierarchical clustering.

## Stereology results datasets:
- Stereo_MULL_BB.csv : a text data file (.csv) of the stereology count results of 173 slides read during this study for Mullus surmuletus individuals sampled in the Bay of Biscay (also available in the Zenodo repository https://zenodo.org/records/8060309)
- Stereo_MULL_EC.csv : a text data file (.csv) of the stereology count results of 294 slides read during this study for Mullus surmuletus individuals sampled in the English Channel (also available in the Zenodo repository https://zenodo.org/records/80603569)


The information contained in these tables are as follows:
  - cell_type: structure identified for one sample point (for the abbreviations, see Heude-Berthelin et al. 2023)
  - idpt: identification number of the sampling point
  - id: unique complex identification number of the sampling point generated by combining the x and y coordinates
  - x: x coordinate of the sampling point
  - y: y coordinate of the sampling point
  - reading: Indicates if the reading data was used to test cellular homogeneity (Homogeneity) or to the sexual maturity phase
  - slideid: identification number of the digitized histological slide that was used for the stereological count. Shares the same 12 first characters with Fish_id found in the Macro_MULL.xlsx file

## Macroscopy results datasets :
- Macro_MULL-BB.xlsx : Excel file (.xlsx) containing measurements of macroscopic parameters for all 103 fish (Mullus surmuletus) sampled during this study in the Bay of Biscay (also available in the Zenodo repository https://zenodo.org/records/8060309)
- Macro_MULL-EC.xlsx : Excel file (.xlsx) containing measurements of macroscopic parameters for all 214 fish (Mullus surmuletus) sampled during this study in the English Channel (also available in the Zenodo repository https://zenodo.org/records/80603569)

The information contained in these tables are as follows:
  - Fish_id: identification of the fish. This id is identical to the name given to the pictures of the full ovaries (Macroscopic_pictures_Data on Zenodo repositories cited above)
  - ICES _Division: International Council for the Exploration of the Sea (ICES) division where the fish was sampled in the Food and agricultural Organization of the United nations (FAO) fishing area 27
  - ICES_statistical_rectangle : Statistical rectangle where the fish was sampled within the FAO fishing area 27
  - Date: date the fish was caught (dd/mm/yyyy)
  - Total_fish_length: total length of the fish (cm)
  - Ungutted_fish_weight: total weight of the fish (g)
  - Otolith_ID: unique identification number given to each sampled fish through the Imagine software used by IFREMER
  - Parasite: presence (Y) or absence (N) of parasite in or on the fish
  - age: age (in years) of the fish after analysis of the fish’s otolith. The IFREMER laboratory of Boulogne-sur-Mer (FRANCE) executed this analysis
  - Visual_maturity : visually estimated maturity, after observation macroscopic criteria of the fish’s gonad with the naked eye, following the WKASMSF (ICES, 2018) scale
  - Liver_weight: liver weight (g)
  - Droite_gonad_weight : gonad weight (g) of right ovary
  - Gauche_gonad_weight : gonad weight (g) of left ovary
  - Sections: number of cross sections sampled for the individual
